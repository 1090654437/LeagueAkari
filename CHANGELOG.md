# 0.0.6 (2023-12-17)

## 新功能

- **自动英雄选择功能**
  - 普通模式自动英雄选择：在可以进行英雄选择时，立即选择预定的英雄。在匹配模式、排位模式或其他任何可以进行主动英雄选择的模式中生效。
  - 随机模式的自动英雄选择：在随机模式中（如大乱斗），可以提供一个英雄列表，当英雄选择台上出现这些英雄时，将自动交换。若同时出现多个目标英雄时，将按照列表中定义的顺序选择。可以设置选择延迟，仅当目标英雄出现在英雄选择台上超过一定时间时，才进行选择。
- **杂项功能**
  - 创建召唤师名称：在**新注册**召唤师时，可以指定召唤师名称。最多可以指定 16 位中文长度。
    _注意：过长的 ID 会导致国服在各类活动中无法绑定大区召唤师信息。名称应控制长度在 14 位及以下长度。_

## 普通调整

- **自动接受信息展示**：现在会展示一个倒计时，表示了距离执行自动接受对局操作的秒数。
